# Analisis de Riesgo Crediticio

## Descripcion
Analisis exploratorio de 31.361 solicitudes de prestamo personal
para identificar los factores asociados al riesgo de impago.

## Dataset
Credit Risk Dataset — descargado via OpenML API

## Preguntas de investigacion
- P1: El ingreso del solicitante influye en el impago?
- P2: La tasa de interes predice el default?
- P3: Los solicitantes jovenes tienen mayor riesgo?
- P4: La carga financiera es el factor mas determinante?

## Hallazgo principal
Los clientes en default destinan el 24% de su ingreso
al prestamo versus el 13% de los que pagan correctamente.

## Tecnologias utilizadas
- Python
- Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn, Scipy
- Google Colab
